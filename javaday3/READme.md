Loops Based Questions 
1. Print Even Numbers from 1 to N
Input N and print all even numbers between 1 and N.

2. Sum of Digits
Input a number and find the sum of its digits.
Example: 1234 → 10

3. Reverse a Number
Input a number and print its reverse.
Example: 1234 → 4321

4. Palindrome Number
Check whether a number is a palindrome.
Example: 121 → Palindrome

5. Armstrong Number
Check whether a number is an Armstrong number.
Example: 153 → 1³+5³+3³ = 153

6. Factorial of a Number
Find the factorial of a given number using a loop.
Example: 5! = 120

7. Prime Number Checker
Check whether a given number is prime or not.

8. Count Even and Odd Digits
Input a number and count how many even and odd digits it contains.
Example: 123456
Even = 3, Odd = 3

9. Multiplication Table Generator
Input a number and print its multiplication table up to 10 or 20.

10. Number Guessing Pattern

Print all numbers from 1 to N:
If divisible by 3 → print "Fizz"
If divisible by 5 → print "Buzz"
If divisible by both → print "FizzBuzz"
Otherwise print the number
